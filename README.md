# AI Industry Lab

An interactive Chinese learning lab for building AI industry and semiconductor investing judgment.

The current release includes:

- 15 core model cards in `output/spine01.html` through `output/spine15.html`
- 14 reference manual pages in `output/m01.html` through `output/m14.html`
- A portable learning plan in `Learn Plan - AI半导体投资.md`
- Product, design, and progress notes for continuing the system

Start from `output/spine01.html`.
